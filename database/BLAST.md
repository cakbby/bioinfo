## 出来ること

塩基配列やアミノ酸配列を様々なデータベースの配列と相同性検索できる。

## 配列間の相同性を検索
### blastn (Nucleotide BLAST)
塩基配列同士で相同性検索する。
Enter Query Sequence に配列（FASTA）を入れ、Choose Search Set の Database から調べたいデータベースを選択する。
