出来ること

二つの分布が異なるか判定できる。具体的には、2群の標本が同一の分布からサンプリングされたか判定できる。ただし、二つの分布の分散が等しいことを仮定している。

内容

方法

① 二つの標本 A, B (それぞれ m, n 個) を混ぜ (合計 m+n 個)、その値を昇順に並べる。その時の順位とカテゴリの関係を覚えておく。同じ値の場合の順位は平均をとることにする（4位と5位の平均 = 4.5）

A = [1, 3, 4, 5]
B = [2, 4, 5, 6]

↓

sample  : [1, 2, 3,   4,   4,   5,   5, 6]
category: [A, B, A,   A,   B,   A,   B, B]
rank    : [1, 2, 3, 4.5, 4.5, 6.5, 6.5, 8]

② 二つのカテゴリの順位の和 TA,TBT_A, T_BTA​,TB​ を求める。

TA=1+3+4.5+6.5=15TB=2+4.5+6.5+8=21 (=8×92−15)\begin{aligned} T_A &= 1 + 3 + 4.5 + 6.5 = 15 \\ T_B &= 2 + 4.5 + 6.5 + 8 = 21 \ (= \frac{8×9}{2} - 15) \end{aligned} TA​TB​​=1+3+4.5+6.5=15=2+4.5+6.5+8=21 (=28×9​−15)​

なお片方の順位が分かれば、全ての順位の和が (m+n)(m+n+1)/2 であることからもう片方の順位もすぐにわかる。

③ 検定に用いる値 U を次のように決める。

U=min⁡(UA,UB)UA=mn+m(m+1)2−TAUB=mn+n(n+1)2−TB\begin{aligned} U &= \min{(U_A, U_B)} \\ U_A &= mn + \frac{m(m+1)}{2} - T_A \\ U_B &= mn + \frac{n(n+1)}{2} - T_B \\ \end{aligned} UUA​UB​​=min(UA​,UB​)=mn+2m(m+1)​−TA​=mn+2n(n+1)​−TB​​

③ n < 9 までは専用の表を用いて検定（p値の算出）を行う。n が 20 以上で中心極限定理が利用できるような状況では、以下の正規分布のもとにUを検定する。

U ∼N(mn2,mn(m+n+1)12)U ~ \sim \mathcal{N} \left( \frac{mn}{2}, \frac{mn(m+n+1)}{12} \right) U ∼N(2mn​,12mn(m+n+1)​)

Python

scipy の stats モジュールを用いる。

from scipy import stats

A = np.random(10)
B = np.random(20)
t = stats.mannwhitneyu(A, B)

仮説

二つの確率変数 x, y が連続な累積分布関数 f, g を持つとする。ここで、"x が y より確率的に小さい" という言葉を

∀a, f(a)>g(a)\forall{a},\ f(a) > g(a) ∀a, f(a)>g(a)

という意味だとする。

今回の帰無仮説は x, y の累積分布が等しい、つまり f=gf = gf=g であり、対立仮説として x が y より確率的に小さい を採用することにする。例えば、確率密度が少し小さい方に平行移動したとしたら、累積分布も左に平行移動するため、確率的に小さくなると考えられる。

今、x が y より確率的に小さいだろうと考えるとき、数値 U を "y (のサンプリング結果) が x (のサンプリング結果) より大きくなった回数" とする。この数値は x が y より小さい値をとりやすい場合低くなるため、U の分布を考え、その期待値 Uˉ\bar{U}Uˉ より低い確率 P(U≤Uˉ)P(U \leq \bar{U})P(U≤Uˉ) をもとに検定することが出来ると考えられる（この値が有意水準より小さいときに帰無仮説を棄却する）。

統計量 U と順位和 T の関係

"y が x より大きくなった回数"に対応する場合、Uの値は順位和との関係を用いて

U=mn+m(m+1)2−TxU = mn + \frac{m(m+1)}{2} - T_x U=mn+2m(m+1)​−Tx​

のように表せる。これは次のように考えれば分かる。Uの値は、"y が x より大きくなることがない"ときに 0 をとるが、あらゆる x と y の結果の並びは U = 0 の並びからいくらか（m以下の回数）交換することで実現できる。また、その時の交換を、下図のような交換に限定しても問題ないと分かる。iii回目の交換で置き換わるyの位置をkik^ikiとおくことにする。

最終的に得られるUの値はこの交換ごとのUの差である ΔU を用いて求めることが出来る。ここで、ΔUの計算においてひとつ前の交換で定まったxの位置(ki−1k^{i-1}ki−1)より右側のみが影響することに注意する（左側はもう動かすことがないようにしているため）。同様に考えれば、TxT_xTx​ の変化 ΔTxi\Delta T_x^iΔTxi​ も右側のみが関係する。そして、ki−1k^{i-1}ki−1より右側では

yy...yxx...xyy...yxx...xyy...yxx...x

の形状が保存されている。したがって、各交換における ΔU は以下の場合を考えれば十分であるといえる。

したがって、ΔU=−ΔTxi\Delta U = - \Delta T_x^iΔU=−ΔTxi​ であり、

U=U0+∑iΔU=−∑iΔTxi=Tx0−TxU = U^0 + \sum_i \Delta U = - \sum_i \Delta T_x^i = T_x^0 - T_x U=U0+i∑​ΔU=−i∑​ΔTxi​=Tx0​−Tx​

となる。ここで、Tx0T_x^0Tx0​ は

Tx0=(m+n)(m+n+1)2−Ty0=m(m+1)+mn2+n(n+1)+mn2−n(n+1)2=mn+m(m+1)2\begin{aligned} T_x^0 &= \frac{(m+n)(m+n+1)}{2} - T_y^0 \\ &= \frac{m(m+1)+mn}{2} + \frac{n(n+1)+mn}{2} - \frac{n(n+1)}{2} \\ &= mn + \frac{m(m+1)}{2} \end{aligned} Tx0​​=2(m+n)(m+n+1)​−Ty0​=2m(m+1)+mn​+2n(n+1)+mn​−2n(n+1)​=mn+2m(m+1)​​

よって、U と TxT_xTx​ の間に

U=mn+m(m+1)2−TxU = mn + \frac{m(m+1)}{2} - T_x U=mn+2m(m+1)​−Tx​

が成り立つと言える。

統計量 U の分布

統計量 U の（帰無仮説の下での）確率分布は、m 個の x と n 個の y とが並ぶ場合の数 = m+nCm\ _{m+n}C_m m+n​Cm​ を、値が U となる場合の数で割ったものに相当する。つまり、

pmn(U)=pˉmn(U) m+nCmp_{mn}(U) = \frac{\bar{p}_{mn}(U)}{\ _{m+n}C_m} pmn​(U)= m+n​Cm​pˉ​mn​(U)​

値が U となる場合の数は以下の漸化式によって求めることが出来る。

pˉmn(U)=pˉ(m−1)n(U)+pˉm(n−1)(U−m)\bar{p}_{mn}(U) = \bar{p}_{(m-1)n}(U) + \bar{p}_{m(n-1)}(U-m) pˉ​mn​(U)=pˉ​(m−1)n​(U)+pˉ​m(n−1)​(U−m)

これは、m+nの個のx,yからなる文字列のうち、最後の文字が x か y かを考えることで求められる。最後の文字が x の場合は、その直前までの(m-1個のxとn個のyからなる)文字列のUの値と同じとなるが、最後の文字が y の場合は、直前の(m個のxとn-1個のyからなる)文字列のUの値は現在のUの値からmを引いたものとなる（最後の y によって +m されるため）。

そして、二項係数に関する次の関係を用いることで、確率の漸化式が得られる。

1 m+nCm=mm+n1 m+n−1Cm−1=1 m+nCn=nm+n1 m+n−1Cn−1\begin{aligned} \frac{1}{\ _{m+n}C_m} &= \frac{m}{m+n} \frac{1}{\ _{m+n-1}C_{m-1}} \\ &= \frac{1}{\ _{m+n}C_n} \\ &= \frac{n}{m+n} \frac{1}{\ _{m+n-1}C_{n-1}} \end{aligned}  m+n​Cm​1​​=m+nm​ m+n−1​Cm−1​1​= m+n​Cn​1​=m+nn​ m+n−1​Cn−1​1​​

∴pmn(U)=mm+np(m−1)n(U)+nm+npm(n−1)(U−m)\therefore p_{mn}(U) = \frac{m}{m+n}p_{(m-1)n}(U) + \frac{n}{m+n}p_{m(n-1)}(U-m) ∴pmn​(U)=m+nm​p(m−1)n​(U)+m+nn​pm(n−1)​(U−m)

この漸化式を解くことで各 m, n に関する U の確率密度を求めることが出来る（論文[1]に表あり）。

また、m, n が大きくなることでこの分布は（中心極限定理から）正規分布に収束する。pmn(U)p_{mn}(U)pmn​(U) の期待値は、y > x となる確率が 1/2 であることから

E[U]=mn2E[U] = \frac{mn}{2} E[U]=2mn​

であり、分散は

Var[U]=mn(m+n+1)12Var[U] = \frac{mn(m+n+1)}{12} Var[U]=12mn(m+n+1)​

となる。分散を求めるには E[U2]E[U^2]E[U2] の漸化式を解けばよい（論文[1]より）。
